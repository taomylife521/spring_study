            用于对象-文档映射的Spring Data MongoDB注解
注　　解                    描　　述
@Document                   标示映射到MongoDB文档上的领域对象
@Id                         标示某个域为ID域
@DbRef                      标示某个域要引用其他的文档，这个文档有可能位于另外一个数据库中
@Field                      为文档域指定自定义的元数据
@Version                    标示某个属性用作版本域
